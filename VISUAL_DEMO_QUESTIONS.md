## How It Works

### 1. User Selects Ledger Type
```
User clicks: Liability â†’ Long-term borrowings â†’ Secured Loans
```

### 2. Component Detects Selection
```typescript
selectedLedgerType = "Secured Loans"
```

### 3. API Call is Made
```http
GET /api/accounting/questions/by_subgroup/?sub_group_1_1=Secured Loans
```

### 4. Questions Render
```json
{
  "count": 6,
  "questions": [
    {
      "id": 1,
      "question": "Name of the lender",
      "field_type": "text",
      "required": true
    },
    {
      "id": 2,
      "question": "Loan amount",
      "field_type": "number",
      "required": true
    },
    {
      "id": 5,
      "question": "Is collateral provided?",
      "field_type": "radio",
      "options": ["Yes", "No"],
      "required": true
    }
  ]
}
```

### 5. User Fills Answers
```typescript
questionAnswers = {
  1: "HDFC Bank",
  2: "5000000",
  3: "8.5",
  4: "60",
  5: "Yes",
  6: "Property"
}
```

### 6. Submit with Ledger
```typescript
POST /api/accounting/masters/ledgers/
{
  "name": "HDFC Bank Loan",
  "category": "Liability",
  "group": "Long-term borrowings",
  "sub_group_1": "Secured Loans",
  "question_answers": {
    1: "HDFC Bank",
    2: "5000000",
    3: "8.5",
    4: "60",
    5: "Yes",
    6: "Property"
  }
}
```
